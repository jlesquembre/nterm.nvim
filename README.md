# nterm.nvim

Warning: WIP

A neovim plugin to interact with the terminal emulator

## Usage

```lua
require'nterm'.init({
  size = 20,
  direction = "horizontal", -- horizontal or vertical
  shell = "fish",
  bg_color = nil,
})
```
